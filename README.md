#### Comando para restaurar a node_modules

```

```

```
npm start
```

#### Comando para rodar com Nodemon

```
npm run dev
```

##### API irá funcionar na porta 3003 por padrão
Você pode configurar a sua porta utilizando a variavel ambiente PORT=
